#### Computer Networks & Protocols

- OSI
  - 物理层
    - 通过传输介质进行比特流传输
  - 数据链路层
    - 数据的流量控制
    - 格式化消息为**帧**
  - 网络层
    - 端对端数据的路由或交换
    - 格式化的消息为**分组**
  - 传输层
    - 可靠透明的数据传输
  - 会话层
    - 执行会话规则
  - 表示层
    - 提供数据格式、编码、加密等
  - 应用层
    - 包括各种协议
